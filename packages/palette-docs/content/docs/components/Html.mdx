---
name: HTML
---

`HTML` renders the given HTML string and wraps it in the typography tags `Sans`
or `Serif` by passing a `fontFamily="sans|serif"`

<Playground>
  <HTML
    fontFamily="sans"
    size={6}
    html={"<h1>H1 Headline</h1><p>Paragraph tag</p><p>Paragraph tag</p>"}
  />
</Playground>

<Playground>
  <HTML
    fontFamily="serif"
    size={4}
    html={"<h1>H1 Headline</h1><p>Paragraph tag</p><p>Paragraph tag</p>"}
  />
</Playground>
